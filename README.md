# Fitness RPG 🎮⚔️

Геймифицированный трекер физической активности. Прокачивай персонажа, выполняя реальные тренировки, следя за питанием и сном.

## Возможности

- 🏃 **Активности** — бег, силовые, борьба, питание, сон, чтение, сжигание калорий
- ⚔️ **Классы персонажа** — 6 базовых классов + 15 комбо-классов с путями развития
- 📊 **Статы** — 12 характеристик на радарной диаграмме
- 🏆 **Достижения** — мифические, секретные, комбо и пента-страйки
- 💎 **Экономика** — кристаллы, магазин снаряжения, бонусы к XP
- ⚡ **Рейды** — боссы с особыми условиями и наградами
- ❤️ **Здоровье** — физическое и ментальное HP с дебаффами
- 🎖️ **Титулы** — выбираемые звания за достижения

## Запуск локально

```bash
# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev

# Собрать для продакшена
npm run build

# Посмотреть продакшен-сборку
npm run preview
```

## Деплой на Vercel

### Способ 1 — через Vercel CLI
```bash
npm i -g vercel
vercel
```

### Способ 2 — через GitHub
1. Запушь репозиторий на GitHub
2. Зайди на [vercel.com](https://vercel.com) → New Project
3. Импортируй репозиторий
4. Vercel автоматически определит Vite и задеплоит

Настройки Vercel (если нужно задать вручную):
- **Framework Preset:** Vite
- **Build Command:** `npm run build`
- **Output Directory:** `dist`

## Технологии

- **React 18** + Vite
- **lucide-react** — иконки
- Хранение данных в `localStorage`
